Stardream.ai

An AI agent for generating VR spaces

The AI takes in a prompt, downloads matching models on Sketchfab, generates textures and wall art, and packages all of the content into a zip file.

The user downloads the zip file and unzips it into an empty Unity project. The content unpacks itself into a VR ready environment.

The whole process takes about 30 minutes.

One half of the AI is a python script that calls into OpenAI apis and the Sketchfab api.

The other half is a Unity C# script that unpacks the AI generated content into a VR world.
